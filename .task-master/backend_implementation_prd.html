ISO 8601 Format: All dates/times should use ISO 8601 format

  • Date only: YYYY-MM-DD (e.g., 2026-01-15)
  • Date with time: YYYY-MM-DDTHH:mm:ssZ (e.g., 2026-01-15T10:30:00Z)
  • Database stores as TIMESTAMP (timezone-aware)
  • Backend should accept both formats and convert appropriately
